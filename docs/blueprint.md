# **App Name**: EduPerform

## Core Features:

- Registration/Login: Secure user authentication system allowing students and faculty to create accounts and log in. Logout to ensure security.
- Student Dashboard: Dashboard to display student performance, highlighting strengths and weaknesses based on uploaded data.
- Assessment Submission: Page for faculty to upload assessments and for students to upload their completed assessments.
- Project Portfolio: Section for students to upload and showcase their projects.
- Feedback Submission: Review page where students can provide likes and dislikes (feedback) on courses or assessments.
- Performance Calculator: Calculator page which uses tool that considers student efficiency and scores from various features to predict performance, providing insights based on those criteria.
- Mini-Games: Integration of two simple browser based games for students to refresh their minds (e.g., a memory game or a quick puzzle).
- Knowledge Exchange: A platform to enable student to connect with the faculty.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which converts to a bright blue (#3399FF), providing a sense of trust and intellect.
- Background color: Desaturated light blue HSL(210, 20%, 95%) converted to hex code #F0F8FF (AliceBlue), creates a calm and professional backdrop.
- Accent color: Analogous blue-violet HSL(240, 60%, 60%) converted to hex code #6666CC, used for interactive elements and highlights to draw attention.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look, ensuring readability and a contemporary feel.
- Use a consistent set of icons to represent different sections of the platform, with a focus on clarity and ease of understanding.
- A clean and structured layout using grid system to maintain visual balance and enhance usability.
- Subtle animations to provide feedback on user interactions (e.g., button presses, loading screens) to improve the user experience.
